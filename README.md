##  Project Overview

</div>

<p>
This project focuses on analyzing pizza sales data using SQL to generate meaningful business insights.
The project includes revenue analysis, customer ordering behavior, top-selling pizzas, peak sales hours, and category-wise performance analysis.
</p>

---

<div align="center">

##  Objectives

</div>

<ul>
  <li>Analyze pizza sales performance</li>
  <li>Calculate total revenue</li>
  <li>Identify top-selling pizzas</li>
  <li>Find peak order hours</li>
  <li>Perform category-wise analysis</li>
  <li>Improve SQL analytical skills</li>
</ul>
